## How to Use

Start with intake and work through each step, or jump directly to the workflow you need:

- **Finding space?** → Start with `01-intake.md`
- **Comparing buildings?** → Jump to `04-compare.md`
- **Getting ready to tour?** → `05-tour-prep.md` + `06-questions.md`
- **Negotiating a deal?** → `07-deal.md`
- **Writing a memo?** → `08-memo.md`

Each workflow file is a standalone prompt you can paste into any AI model or run directly in Claude Code.
