# PromptedCRE — Landing Page

Marketing site for [promptedcre.com](https://promptedcre.com). Built with Next.js 14, shadcn/ui, and Tailwind CSS. Deployed via Vercel.

## Dev Setup

```bash
cd landing
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed automatically on push to `main` via Vercel. Root directory is set to `landing/`.

## Stack

- [Next.js 14](https://nextjs.org)
- [shadcn/ui](https://ui.shadcn.com)
- [Tailwind CSS](https://tailwindcss.com)
- [Vercel](https://vercel.com)
